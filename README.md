## 一、这是什么

一个**完全免费的网站上线方法**。不需要买服务器，不需要备案，改完代码自动更新，数据不会丢。

你只需要把代码推到 GitHub，剩下的三个免费服务帮你搞定：
- **Render** → 运行你的代码（像一台不要钱的服务器）
- **Supabase** → 存你的数据（像免费的数据库硬盘）
- **GitHub Actions** → 自动干活（每次改代码自动更新 + 防止网站睡死）

> 不是 GitHub Pages 那种只能放图片文字的静态页面——它能跑真实的登录、数据库、后端逻辑。

## 四、怎么用

1. 把本仓库的 `render.yaml` 和 `.github/workflows/` 复制到你项目里
2. 改两处名字（render.yaml 里的项目名 + keep-alive.yml 里的网址）
3. 注册 Render 和 Supabase（都用 GitHub 一键登录）
4. Supabase 上创建一个免费数据库，复制连接地址
5. 代码推到 GitHub，Render 自动部署
6. 把数据库地址填进 Render 的环境变量
7. 搞定 🎉

详细步骤 → [SKILL.md](./SKILL.md)

## ⚠️ 免费的东西有什么限制

| 服务 | 限制 | 有啥影响 |
|------|------|---------|
| Render | 15 分钟没人访问就休眠 | 下一个人打开要等半分钟（本方案已用 GitHub Actions 每 14 分钟戳一次解决） |
| Render | 磁盘是临时的，重启就清空 | **不要往磁盘存数据**，数据库只用 Supabase |
| Supabase | 最多存 500MB | 够存几万条记录，小项目用不完 |
| Supabase | 90 天不活跃会暂停 | 会提前发邮件提醒你 |
| GitHub Actions | 仓库 60 天没新提交，定时任务停 | 随便改点东西 push 一下就恢复 |
